# Contact Manager Scaffold

This file describes the scaffold created for the Contact Management Mini-App.

What I created:

- Project config: `package.json`, `vite.config.ts`, `tsconfig.json`
- App entry: `index.html`, `src/main.tsx`, `src/index.css`
- Components (skeletons): `src/components/ContactList.tsx`, `src/components/ContactForm.tsx`, `src/components/ContactItem.tsx`
- Types: `src/types.ts`
- Mock data: `src/data/contacts.json`
